## 📖 Deskripsi
ResepPuput adalah aplikasi Android berbasis Jetpack Compose yang membantu pengguna mencari resep masakan dengan mudah. Aplikasi ini menampilkan daftar resep lengkap dengan gambar, bahan, bumbu, serta langkah-langkah memasak secara detail.
Aplikasi dibuat sebagai tugas Evaluasi Tengah Semester (ETS) mata kuliah Pemrograman Perangkat Bergerak.

---

## ✨ Fitur Utama

- 📋 Menampilkan daftar resep (LazyColumn & Card UI)
- 📄 Detail resep (gambar, deskripsi, bahan, bumbu, langkah)
- 🔍 Pencarian resep berdasarkan nama
- 🎯 Highlight hasil pencarian
- ⏳ Loading state (simulasi loading)
- ❌ Empty state (resep tidak ditemukan)
- 🎨 UI modern, responsif, dan user-friendly (Material 3)

## 📲 Cara Install Aplikasi

1. Download file APK dari link di atas
2. Pindahkan file ke perangkat Android
3. Buka file APK
4. Jika muncul peringatan, aktifkan:
   > **"Install from unknown sources"**
5. Klik **Install**
6. Tunggu hingga selesai
7. Aplikasi siap digunakan 🎉
